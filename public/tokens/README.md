# Token Images Directory

Place all token logo images in this directory.

Required files:
- eth.png (Ethereum)
- afrox.png (AfroDex)
- afdlts.png (AfroDex Labs Token)
- pfarm.png (PFARM)
- free.png (FREE Coin)
- plaas.png (PLAAS)
- lwb.png (Living Without Borders)
- t1c.png (Travel1Click)
- bct.png (Bitcratic)
- usdt.png (Tether)
- busd.png (Binance USD)
- dai.png (DAI)
- weth.png (Wrapped Ether)
- empty-token.png (Placeholder for unlisted tokens)

All images should be:
- Square aspect ratio (recommended: 64x64 or 128x128)
- PNG format with transparency
- Optimized for web
