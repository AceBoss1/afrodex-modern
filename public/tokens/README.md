# Token Images Directory

Place all token logo images in this directory.

## Required Files

| File | Token |
|------|-------|
| `eth.png` | Ethereum |
| `afrox.png` | AfroDex |
| `afdlts.png` | AfroDex Labs Token |
| `pfarm.png` | PFARM |
| `free.png` | FREE Coin |
| `plaas.png` | PLAAS |
| `lwb.png` | Living Without Borders |
| `t1c.png` | Travel1Click |
| `bct.png` | Bitcratic |
| `usdt.png` | Tether |
| `busd.png` | Binance USD |
| `dai.png` | DAI |
| `weth.png` | Wrapped Ether |
| `empty-token.png` | Placeholder for unlisted tokens |

## Image Specifications

- **Format**: PNG with transparency
- **Size**: 64x64 or 128x128 pixels (square)
- **File size**: Less than 50KB each
- **Background**: Transparent or matching token branding

## Creating empty-token.png

If you don't have an empty-token.png placeholder, create a simple circle with a question mark or the text "?" to serve as a fallback for unlisted tokens.
